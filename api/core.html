class pypipr.PupilSeries(time_data: ndarray[tuple[Any, ...], dtype[number]], size: ndarray[tuple[Any, ...], dtype[number]], stimuli_list: list[tuple[float, float]], name: str | None = None)[source]

Bases: PupilMeasurement

A class representing a series of pupil measurements with multiple light stimuli.

This class extends PupilMeasurement to handle multiple light stimuli events and provides functionality to split the series into individual measurements.
