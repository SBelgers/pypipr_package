Pupillary Light Response Simulation

This notebook demonstrates how to simulate realistic pupillary light responses using the simulate_pupil_measurement function:

  • Creating basic simulated traces

  • Customizing simulation parameters

  • Understanding pupil response phases

  • Comparing different stimulus conditions

  • Adding noise and realistic variations

# Import necessary libraries
import matplotlib.pyplot as plt
import numpy as np

import pypipr

1. Basic Pupil Response Simulation

The simulate_pupil_measurement function creates a realistic pupillary light response with different phases:

  • Baseline: Stable pupil size before light onset

  • Latency: Brief delay after light onset (Currently undefined)

  • Constriction: Rapid pupil size reduction

  • Sustained: Maintained constriction during light exposure

  • Redilation: Exponential recovery after light offset

# Create a basic simulated pupil measurement with default parameters
pupil_measurement = pypipr.simulate_pupil_measurement()

# Plot the simulated pupil response
fig, ax = plt.subplots(figsize=(12, 6))
pupil_measurement.plot(ax=ax, color="blue", linewidth=2, label="Pupil Size")
pupil_measurement.plot_light_stimulus(
    ax=ax, color="yellow", alpha=0.3, label="Light Stimulus"
)
ax.set_title("Simulated Pupil Response")
ax.legend()

2. Customizing Simulation Parameters

The simulation function allows you to control various aspects of the pupil response:

Timing Parameters:

  • t_start, t_end: Overall simulation time window

  • t_light_onset, t_light_offset: Light stimulus timing

  • num_samples: Number of data samples

# Create a simulation with custom timing parameters
pupil_measurement = pypipr.simulate_pupil_measurement(
    t_start=0.0, t_end=16.0, t_light_onset=2.0, t_light_offset=6.0, num_samples=1000
)

# Plot the simulated pupil response
fig, ax = plt.subplots(figsize=(12, 6))
pupil_measurement.plot(ax=ax, color="blue", linewidth=2, label="Pupil Size")
pupil_measurement.plot_light_stimulus(
    ax=ax, color="yellow", alpha=0.3, label="Light Stimulus"
)
ax.set_title("Simulated Pupil Response with Custom Timing")
ax.legend()

<matplotlib.legend.Legend at 0x21971659e50>

3. Customizing Response Parameters

You can control the shape and magnitude of the pupil response by adjusting physiological parameters:

Response Shape Parameters:

  • baseline_param_y0: Initial pupil diameter (typical range: 4-8 mm)

  • constrict_param_cv: Constriction velocity - how fast the pupil contracts (negative values, typical: -20 to -30 mm/s)

  • sustained_param_m: Sustained phase slope - gradual change during light exposure

  • sustained_param_c: Sustained phase offset

  • redilation_param_s: Redilation amplitude scaling

  • redilation_param_k: Redilation rate constant (negative values for exponential recovery)

  • redilation_param_p: Redilation phase parameter

Let’s create a simulation with custom response parameters:

# Create a simulation with custom response parameters
pupil_measurement = pypipr.simulate_pupil_measurement(
    baseline_param_y0=7.5,  # Larger baseline pupil size
    constrict_param_cv=-24.0,  # Moderate constriction speed
    sustained_param_m=0.1,  # Slight upward drift during sustained phase
    redilation_param_k=-0.5,  # Redilation rate
    redilation_param_p=6.2,  # Redilation shape parameter
)

# Plot the simulated pupil response
fig, ax = plt.subplots(figsize=(12, 6))
pupil_measurement.plot(ax=ax, color="blue", linewidth=2, label="Pupil Size")
pupil_measurement.plot_light_stimulus(
    ax=ax, color="yellow", alpha=0.3, label="Light Stimulus"
)
ax.set_title("Simulated Pupil Response with Custom Parameters")
ax.set_ylim(2, 8)
ax.legend()

4. Comparing Different Stimulus Conditions

Let’s simulate and compare different light stimulus conditions to understand how parameters affect the pupil response:

# Create simulations with different stimulus durations and intensities
red_stim = pypipr.simulate_pupil_measurement(
    constrict_param_cv=-2.9,
    t_peak_constrict_rel_to_light_onset=0.22,
    redilation_param_k=-0.8,
)
blue_stim = pypipr.simulate_pupil_measurement()

# Plot comparison
fig, ax = plt.subplots(1, 1, figsize=(12, 10))

# Short stimulus
red_stim.plot(ax=ax, color="red", linewidth=2, label="Red stimulus")
red_stim.plot_light_stimulus(ax=ax, color="orange", alpha=0.3)

# Long stimulus
blue_stim.plot(ax=ax, color="blue", linewidth=2, label="Blue stimulus")
blue_stim.plot_light_stimulus(ax=ax, color="yellow", alpha=0.3)
ax.set_title("Light Stimulus comparison")
ax.legend()

print("Stimulus comparison:")
print(f"Short stimulus - Max constriction: {np.nanmin(red_stim.get_size()):.2f} mm")
print(f"Long stimulus - Max constriction: {np.nanmin(blue_stim.get_size()):.2f} mm")

5. Adding Realistic Noise and Variations

Real pupillometry measurements contain noise from various sources. The simulation function allows you to add realistic noise components:

fig, ax = plt.subplots(1, 1, figsize=(12, 12))
np.random.seed(0)
# Create simulations with different noise levels
clean_signal = pypipr.simulate_pupil_measurement(noise_std=0.0, trend_std=0.0)
clean_signal.plot(ax=ax, color="green", linewidth=1.5, label="No noise")
clean_signal.plot_light_stimulus(ax=ax, color="yellow", alpha=0.3)

noisy_signal = pypipr.simulate_pupil_measurement(noise_std=0.001, trend_std=0.001)
noisy_signal.plot(ax=ax, color="orange", linewidth=1.5, label="Moderate noise")

very_noisy_signal = pypipr.simulate_pupil_measurement(noise_std=0.005, trend_std=0.005)
very_noisy_signal.plot(ax=ax, color="red", linewidth=1.5, label="High noise")

ax.legend()
